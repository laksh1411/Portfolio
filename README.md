# Personal Portfolio Website

Welcome to my personal portfolio repository! This project showcases my skills, experience, and the various projects I've built as a Data Scientist and AI Engineer.

## 🚀 Live Demo

You can view the live, deployed version of my portfolio here:  
**[👉 View Live Portfolio](https://lakshsaluja.vercel.app/)**  
*(Deployment automatically managed by Vercel)*

## 🛠️ Built With
- **HTML5:** Semantic structure and organization
- **CSS3:** Custom styling, glassmorphism effects, and animations. No external UI frameworks were used to demonstrate core CSS abilities!
- **JavaScript (Vanilla):** Scroll animations, smooth typing effects, and interactive form handling.
- **Vercel:** Hosting platform

## ✨ Features
- **Modern UI/UX:** Dark mode aesthetics with glassmorphism (frosted glass) panels.
- **Responsive Design:** Fully fluid and behaves beautifully on phones, tablets, and desktop monitors.
- **Dynamic Animations:** Interactive cursor glow, scroll-reveals (Intersection Observer API), and glitch text effects.
- **Interactive Metrics:** Visual representation of problem-solving statistics and GitHub data.

## 🏃‍♂️ How to Run Locally

If you'd like to run this codebase on your own machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/laksh1411/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in any web browser, or use an extension like VS Code's **Live Server** to spin up a local development server.
